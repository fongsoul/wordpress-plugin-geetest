# WordPress Plugin Geetest

帮助你的 WordPress 抵御恶性骚扰。

---

## 下载并配置

+ 下载最新 [Geetest](https://github.com/mousin/WordPress-Plugin-Geetest/archive/master.zip)
+ 后台菜单 "插件" 上传插件并启用
+ 后台菜单 "极验设置" 配置[极验官网](https://www.zybuluo.com/cmd/)申请的 ID, KEY
